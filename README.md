## Description

API para criação e controle de despesas (CRUD) com Autenticação JWT, API feita em Nest.js.

## Installation

```bash
$ npm install
```

## Documentation
- [Swagger](http://localhost:3000/api#/) (Local)

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
